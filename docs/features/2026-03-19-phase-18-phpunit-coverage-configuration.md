# Phase 18 - PHPUnit 11 Coverage Configuration

Date: 2026-03-19
Status: Completed

## Summary

This update finalizes PHPUnit setup for the package with a PHPUnit 11.5-compatible XML configuration and reproducible coverage commands.

## Changes

1. Updated `phpunit.xml` to use PHPUnit 11.5 schema.
2. Migrated source selection to `<source>` include/exclude rules for `src/`.
3. Added coverage reports:
- Text report to stdout.
- Clover XML report at `build/logs/clover.xml`.
- HTML report at `build/coverage`.
4. Added Composer scripts in `composer.json`:
- `composer test`
- `composer test:unit`
- `composer test:integration`
- `composer test:coverage`

## Commands

```bash
composer test
composer test:unit
composer test:integration
composer test:coverage
```

## Notes

- Coverage requires a PHP coverage driver (Xdebug or PCOV).
- `XDEBUG_MODE=coverage` is configured in `phpunit.xml` for consistency.
- Generated coverage artifacts are expected under `build/`.

## Validation Expectations

- Unit and Integration suites execute from configured test suites.
- Coverage reports are generated by `composer test:coverage`.
- Source coverage scope remains limited to `src/`.
